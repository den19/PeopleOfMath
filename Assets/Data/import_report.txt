Portrait import — run in Unity Editor
=====================================

Use: PeopleOfMath → Import Real Portraits (replace placeholders)

This replaces dev placeholders, downloads from Wikidata P18 / Commons / Wikipedia,
then links sprites to ScriptableObjects automatically.

After import, check counts in Console log (OK >=2 / partial / skip).
Manual files: Assets/Resources/Portraits/{id}/01.jpg, 02.jpg
Then: PeopleOfMath → Link Portraits From Folders

Last automated report will be overwritten on next import.
